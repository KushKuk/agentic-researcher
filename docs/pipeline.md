# Pipeline Documentation

## Overview

The pipeline coordinates execution between agents and tools.

It acts as the main entry point for workflows triggered by API requests.

---

## Current Pipeline

1. Receive research topic
2. Initialize planner agent
3. Execute planner
4. Planner calls research tool
5. Results returned
